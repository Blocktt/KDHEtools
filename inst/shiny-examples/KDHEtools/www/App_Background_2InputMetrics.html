Input metrics and scoring formulae

There are five input metrics in the KS MMI (Table 1). Each has its own random forest model and set of predictor variables. Predictor variables are used in the random forest analysis to account for natural gradient effects on the metrics. For each site, the random forest model looks at site characteristics through decision 'trees' and arrives at a predicted metric values for that site based on the site characteristics, as captured by the predictor values.

The random forest adjusted metric value = observed value â€“ predicted value

The residual (adjusted) value can be thought of as the variation remaining after accounting for natural factors.

The adjusted metric residuals were standardized to a 100-point scale such that higher values represented better conditions. The metric standardization was an interpolation of metric values between the 5th and 95th percentiles of each metric distribution, using the scoring formulas in Table 1. If metric values met the reference expectation (predicted = observed), they received scores in the 50-60 range. MMI values were calculated as the average of the individual metric scores.
