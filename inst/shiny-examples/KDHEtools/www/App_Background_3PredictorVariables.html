Metric Adjustment Factors

The Kansas Macroinvertebrate MMI contains five metrics that are adjusted based on site-specific natural factors including flow estimates from Perry et al. 2002, StreamCat estimates (StreamCat), and Level III Ecoregion. To retrieve data for a site, the user must first obtain the appropriate COMID by conducting a spatial join between the site location (i.e., latitude/longitude) and the NHDPlus V2 flowline layer (outside of the app). Once each input site has been assigned a COMID, the user can use the Prepare Data tab to obtain COMID-specific data. Metric adjustment factor data are required for the Calculate MMI input file. See Table 2 for additional information on the natural variables. See Table 3 (in Calculate MMI tab) for necessary input fields. For more information on the derivation of these metric adjustment factors, please read the Kansas Macroinvertebrate MMI report. Note, only COMIDs from the state of Kansas can be used. If the user would like COMIDs from other, nearby states, please contact Ben Block (Ben.Block@tetratech.com) and he can update the available list.
